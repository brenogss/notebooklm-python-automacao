# 💬 Prompts utilizados

Durante o uso do NotebookLM, foram elaborados prompts para explorar os materiais adicionados à base de conhecimento.

## Python

- O que é automação com Python e quais são suas principais aplicações?
- Quais bibliotecas Python são mais utilizadas para automação?

## APIs

- Como consumir uma API REST utilizando a biblioteca Requests?
- Qual a importância das APIs em projetos de automação?

## Automação Web

- Quais são as diferenças entre Selenium e Playwright?
- Em quais situações devo utilizar cada uma dessas bibliotecas?

## Manipulação de Dados

- Como automatizar planilhas do Excel utilizando Pandas e OpenPyXL?

## IA Aplicada

- Como a Inteligência Artificial pode aumentar a produtividade em projetos de automação com Python?
