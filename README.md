## 📖 Sobre

Este repositório foi desenvolvido como parte do desafio da DIO utilizando o NotebookLM.

O objetivo é criar uma base de conhecimento sobre Python para automação, utilizando documentação oficial e materiais de referência para apoiar os estudos.

---

## 🎯 Objetivos

- Aprender a utilizar o NotebookLM.
- Organizar uma base de conhecimento personalizada.
- Explorar Python aplicado à automação.
- Utilizar IA como ferramenta de apoio aos estudos.
